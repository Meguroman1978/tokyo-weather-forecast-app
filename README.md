# 東京都天気予報アプリ

東京的都内の市区町村別の天気予報を確認できるウェブアプリケーションです。

## 機能

- **市区町村選択**: 東京都内の全62市区町村（23特別区、市部、郡部）を選択可能
- **現在天気**: 温度、天気描述、体感温度、湿度、風速、気圧を表示
- **5日間予報**: 選択した地域の5日間の天気予報を表示
- **レスポンシブデザイン**: スマートフォンやタブレットに対応

## 必要なもの

- OpenWeatherMap APIキー（無料で取得可能）

## セットアップ手順

### 1. APIキーの取得

1. [OpenWeatherMap](https://openweathermap.org/api) にアクセス
2. 無料アカウントを作成
3. APIキーを取得

### 2. APIキーの設定

1. `app.js` ファイルを開く
2. 8行目の `YOUR_API_KEY_HERE` を取得したAPIキーに変更

```javascript
const API_KEY = 'あなたのAPIキー'; // 実際のAPIキーに変更
```

### 3. アプリケーションの実行

1. `index.html` ファイルをブラウザで開く
2. 市区町村を選択
3. 天気情報を表示

## 使用方法

1. **市区町村選択**: ページ上部のドロップダウンメニューから目的の価格町村を選択
2. **天気確認**: 選択と同時に現在天気が自動表示
3. **データ更新**: 「🔄 更新」ボタンで最新の天気に更新
4. **5日間予報**: 現在天気の，下に5日間予報が表示
