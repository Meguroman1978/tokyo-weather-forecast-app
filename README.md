# 東京都天気予報アプリ

東京的都内の市区町村別の天気予報を確認できるウェブアプリケーションです。

## 機能

- **市区町村選択**: 東京都内の全62市区町村（23特別区、市部、郡部）を選択可能
- **現在天気**: 温度、天気描述、体感温度、湿度、風速、気圧を表示
- **5日間予報**: 選択した地域の5日間の天気予報を表示
- **レスポンシブデザイン**: スマートフォンやタブレットに対応

## 🔧 必要なもの

- ** NOTHING! ** このアプリケーションは完全に無料で、APIキーや登録は一切不要です！

## 🚀 セットアップ手順

### セットアップは不要です！

1. **ファイルをブラウザで開く**: `index.html` ファイルをダブルクリック
2. **市区町村を選択**: ドロップダウンメニューから目的の価格町村を選択
3. **天気確認**: 自動的に天気情報が表示

## 使用方法

1. **市区町村選択**: ページ上部のドロップダウンメニューから目的の価格町村を選択
2. **天気確認**: 選択と同時に現在天気が自動表示
3. **データ更新**: 「🔄 更新」ボタンで最新の天気に更新
4. **5日間予報**: 現在天気の，下に5日間予報が表示
